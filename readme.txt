== Description ==

Ayet referansları bulup linklere dönüştürür ve seçeneğe baglı ayet araçipucu içinde gösterir.

== Installation ==

1. İndirdiğin zip dosyasında bulunan incilinfo klasörünü `/wp-content/plugins/` klasörüne yükleyiniz.
1. Eklentiler sayfasından eklentiyi etkinleştiriniz.
